capitle
=======
The capital city guessing game.


Development
-----------
```
./server.sh
```

Open http://localhost:8000/index.html


Releasing
---------
If you're on a new server run `boostrap.sh`

```
./install.sh
```


Testing
-------
Open http://localhost:8000/index.html?test=1 and check the console.
